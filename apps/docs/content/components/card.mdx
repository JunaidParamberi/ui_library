## Usage

Compose `Card`, `CardHeader`, `CardTitle`, `CardBody`, and `CardFooter` to build a bordered
content surface. Set `interactive` when the whole card is clickable (adds a hover shadow and
pointer cursor) and `dashed` for an empty/placeholder slot.

## Accessibility

`Card` renders a plain `<div>` and carries no implicit semantics, even with `interactive`
styling applied. If a card acts as a link or button, nest a real `<a>`/`<button>` element
inside it (or wrap the whole card in one) so it's reachable by keyboard and exposed to
assistive tech with the correct role — the `interactive` variant only affects visual
affordance, not behavior.
