# ThemeToggle

Controlled light/dark toggle button. Has no side effects of its own — the consumer applies the
theme (e.g. via `document.documentElement.classList`).

## Usage

Pass the current `theme` and an `onThemeChange` handler that applies the theme (e.g. toggling
a `dark` class on `<html>` and persisting the choice). `ThemeToggle` is a thin, controlled
wrapper around `Button` (`variant="ghost"`, `size="icon"` by default) — it renders a moon icon
in light mode and a sun icon in dark mode, and accepts any other `ButtonProps` except `onClick`
and `children`.

## Accessibility

Renders a native icon-only `<button>` with an automatic `aria-label` ("Switch to dark theme" /
"Switch to light theme") describing the action that will happen on click, so screen reader
users don't need to infer meaning from the icon alone. Keyboard-activatable with Enter/Space
like any button.
