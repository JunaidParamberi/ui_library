# ThemeToggle

Controlled light/dark toggle button. Has no side effects of its own — the consumer applies the theme (e.g. via `document.documentElement.classList`).

## Props

| Prop | Type | Default |
|------|------|---------|
| `theme` | `"light" \| "dark"` | required |
| `onThemeChange` | `(next: "light" \| "dark") => void` | required |

Extends all `ButtonProps` except `onClick` and `children`.
