# AreaChart

Lightweight dataviz primitives rendered as inline SVG — no charting dependency: `AreaChart`
(line + filled area) and `MiniBars` (sparkline-style bars).

## Usage

Pass `data` as an array of `{ x, y }` points to `AreaChart` or plain numbers to `MiniBars`.
Both are intentionally minimal (no axes, tooltips, or legends) — use them for compact trend
indicators next to a `KPICard` or in a dashboard tile, not as a full charting solution. Adjust
`width`/`height` to fit the surrounding layout.

## Accessibility

Both render plain inline SVG with no default `role` or text alternative, since they're
typically paired with a visible metric (e.g. next to a `KPICard`'s value) that already conveys
the same information. If a chart is the sole source of information, add `role="img"` and an
`aria-label` summarizing the trend.
