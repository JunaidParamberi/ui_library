## Usage

`Checkbox` accepts an optional `label` that renders a connected, clickable `<label>`. Use
`checked="indeterminate"` for a "select all" checkbox representing a partial selection. The
same file also exports `Switch` (boolean on/off toggle) and `RadioGroup`/`RadioGroupItem` for
mutually-exclusive choices — reach for `Switch` for settings that apply immediately, and
`Checkbox` for form selections that need explicit submission.

## Accessibility

Built on Radix `Checkbox`, exposing `role="checkbox"` with `aria-checked` (including
`"mixed"` for indeterminate) and toggling with Space. The `label` prop is wired to the input
via matching `id`/`htmlFor`, so clicking the label text also toggles the control. `Switch` uses
Radix's switch role and behaves identically for keyboard users.
