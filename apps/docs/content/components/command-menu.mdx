# CommandMenu

A ⌘K-style command palette with search-as-you-type filtering and grouped items, built on
`cmdk` inside a `Dialog`.

## Usage

Control `open`/`onOpenChange` from a keyboard shortcut (typically ⌘K/Ctrl+K) as well as a
visible trigger. Pass `items` with an `id`, `label`, `onSelect` handler, and optional `group` —
items sharing a `group` are rendered under one heading, ungrouped items fall under "Commands".
Use it for global, searchable navigation/actions, not as a replacement for a regular menu tied
to one object.

## Accessibility

Renders inside a `Dialog`, so it inherits focus trapping, Escape-to-close, and a
screen-reader-only title (`"Command menu"`). The search input is auto-focused on open; results
update live as you type and are navigable with arrow keys, with Enter activating the
highlighted item — the standard `cmdk` combobox pattern.
