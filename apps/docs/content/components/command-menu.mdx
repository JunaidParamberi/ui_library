# CommandMenu

A ⌘K-style command palette with search-as-you-type filtering and grouped items, built on `cmdk` inside a `Dialog`.

## Props

| Prop | Type | Default |
|------|------|---------|
| `open` | `boolean` | required |
| `onOpenChange` | `(open: boolean) => void` | required |
| `items` | `{ id: string; label: string; onSelect: () => void; group?: string }[]` | required |
| `className` | `string` | — |

Items sharing the same `group` are rendered under one heading; ungrouped items fall under "Commands".
