## Usage

Compose `DropdownMenu` > `DropdownMenuTrigger` (`asChild` around a `Button`) > `DropdownMenuContent`
with `DropdownMenuItem`s, grouped with `DropdownMenuLabel` and separated with
`DropdownMenuSeparator`. Use for a short list of actions tied to an object (row actions, an
overflow menu) — for navigation between views prefer `Tabs`, and for a single confirmation
prefer `Dialog`.

## Accessibility

Built on Radix `DropdownMenu`, implementing the ARIA menu pattern: the trigger opens the menu
with Enter/Space/ArrowDown, items are navigable with arrow keys and typeahead, and Escape
closes the menu and returns focus to the trigger. Disabled items are marked with
`aria-disabled` and are skipped during keyboard navigation.
