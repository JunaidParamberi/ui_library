## Props

### DataTable

| Prop | Type | Default |
|------|------|---------|
| `columns` | `{ key: string; header: string; render?: (row: T) => ReactNode; sortable?: boolean }[]` | required |
| `data` | `T[]` | required |
| `getRowId` | `(row: T) => string` | required |
| `className` | `string` | — |

Clicking a sortable header cycles ascending → descending → unsorted. All Table primitives forward `ref`, merge `className` via `cn()`, and spread remaining props.
