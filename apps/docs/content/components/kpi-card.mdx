## Usage

Pass a `kpi` object with `label`, `value`, and optionally `delta`, `trend`
(`"up"`/`"down"`/`"flat"`), and `sub`. `trend="up"` renders a success-colored badge with an
up arrow; `"down"` and `"flat"` render a neutral outline badge. Use `delay` to stagger the
fade-up entrance animation when rendering several `KPICard`s in a row (e.g. `delay={index}`).

## Accessibility

Renders as a `Card` (a plain `<div>`) with text content in normal reading order — label,
value, then delta/subtitle — so it's readable by screen readers without extra markup. The
trend arrow icons are marked `aria-hidden`; the delta text itself (e.g. `"+12%"`) carries the
meaning, so don't rely on the arrow or badge color alone to convey direction.
