## Props

### Progress

| Prop | Type | Default |
|------|------|---------|
| `value` | `number` | required, clamped to 0-100 |
| `className` | `string` | — |

### HealthRing

| Prop | Type | Default |
|------|------|---------|
| `value` | `number` | required, clamped to 0-100 |
| `size` | `number` | `64` |
| `strokeWidth` | `number` | `6` |
| `label` | `string` | — |
| `className` | `string` | — |
