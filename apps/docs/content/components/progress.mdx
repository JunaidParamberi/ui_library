## Usage

Use `Progress` for a linear, determinate task (uploads, multi-step completion) and
`HealthRing` where a compact radial summary fits better (dashboard tiles, score widgets). Both
clamp `value` to 0–100 — you don't need to clamp it yourself. Always pass an `aria-label`
describing what's progressing, since neither component renders visible text by default.

## Accessibility

`Progress` is built on Radix `Progress`, which exposes `role="progressbar"` with
`aria-valuenow`/`aria-valuemin`/`aria-valuemax` so assistive tech can announce completion.
`HealthRing` is a custom SVG with no built-in ARIA — pass `aria-label` (and consider
`role="img"`) yourself if you use it standalone rather than as a decorative companion to
visible text.
