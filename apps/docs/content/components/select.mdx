## Usage

Compose `Select`, `SelectTrigger`, `SelectValue`, `SelectContent`, and `SelectItem`; group
related options with `SelectGroup` + `SelectLabel`, and divide sections with
`SelectSeparator`. Always pass an accessible name to `SelectTrigger` via `aria-label` or a
visible `Field` label. Use `inputSize="lg"` to match larger form layouts.

## Accessibility

Built on Radix `Select`, which implements the ARIA listbox pattern: the trigger is a native
`<button>` opened with Enter/Space/Arrow keys, options are navigable with arrow keys and
typeahead, and the selected value is announced. Selection updates via keyboard or pointer both
close the popover and return focus to the trigger.
