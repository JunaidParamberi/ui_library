# Avatar

Displays a user's profile image with a graceful fallback to initials when the image fails to
load or hasn't loaded yet.

## Usage

Compose `Avatar`, `AvatarImage`, and `AvatarFallback`. `AvatarFallback` renders automatically
while the image loads and if it errors, so always provide it (typically the user's initials).
Use the `size` variant (`sm`/`default`/`lg`) to match the surrounding density — e.g. `sm` in a
table row, `lg` in a profile header.

## Accessibility

Built on Radix `Avatar`. Always pass a meaningful `alt` to `AvatarImage` describing the person
(e.g. `"Jane Doe"`), not generic text like `"avatar"`. The fallback initials are plain text and
are announced by screen readers when the image isn't rendered.
