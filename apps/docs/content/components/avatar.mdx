# Avatar

Displays a user's profile image with a graceful fallback to initials when the image fails to load or hasn't loaded yet.

## Props

### Avatar

| Prop | Type | Default |
|------|------|---------|
| `size` | `"sm" \| "default" \| "lg"` | `"default"` |
| `className` | `string` | — |

All other props are forwarded to the underlying element. Also exported: `AvatarImage`, `AvatarFallback`.
