# AppShell

Application layout shell: a fixed-width sidebar column, a sticky topbar row, and a scrollable content area. The sidebar collapses under the `md` breakpoint.

## Props

| Prop | Type | Default |
|------|------|---------|
| `sidebar` | `ReactNode` | required |
| `topbar` | `ReactNode` | required |
| `sidebarOpen` | `boolean` | `true` |
| `sidebarWidth` | `number` | `256` |
| `children` | `ReactNode` | required |

`AppShell` holds no internal collapse state — the consumer controls `sidebarOpen`, matching the library's customization contract.
