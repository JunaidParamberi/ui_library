# AppShell

Application layout shell: a fixed-width sidebar column, a sticky topbar row, and a scrollable
content area. The sidebar collapses under the `md` breakpoint.

## Usage

Pass `sidebar` and `topbar` as `ReactNode` slots and put your page content as `children`.
`AppShell` holds no internal collapse state — the consumer controls `sidebarOpen`, matching the
library's customization contract, so wire it up to your own navigation/menu toggle. Adjust
`sidebarWidth` to fit your navigation content.

## Accessibility

`AppShell` already wraps the sidebar in `<aside>`, the topbar in `<header>`, and content in
`<main>`, giving assistive tech a meaningful landmark structure out of the box. Wrap the
`sidebar` node's navigation links in a `<nav>` with an `aria-label` (e.g. `"Main"`) so it's
distinguishable from other landmarks when there's more than one on the page.
