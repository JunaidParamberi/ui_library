# Icons

A curated `Record<string, LucideIcon>` re-export of the `lucide-react` icons already used across the library — one stable import surface instead of importing `lucide-react` directly.

<StorybookLink package="ui" story="components-icons" />

<ComponentPreview>

</ComponentPreview>

## Usage

```tsx
import { Icons } from "@manpowerhub/ui";

<Icons.Search className="size-4" />
```

Each value is a `LucideIcon` component and accepts all standard `lucide-react` props (`className`, `size`, `strokeWidth`, etc).
