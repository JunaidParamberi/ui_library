# Icons

A curated `Record<string, LucideIcon>` re-export of the `lucide-react` icons already used
across the library — one stable import surface instead of importing `lucide-react` directly.

## Usage

```tsx
import { Icons } from "@manpowerhub/ui";

<Icons.Search className="size-4" />
```

Each value is a `LucideIcon` component and accepts all standard `lucide-react` props
(`className`, `size`, `strokeWidth`, etc). `Icons` is a namespace re-export rather than a
single component, so there's no props table or states gallery for it — see each icon's own
props via `lucide-react`.

## Accessibility

Icons are visual-only by default and should be marked `aria-hidden` when paired with visible
text (as in the preview above). When an icon is the only content of an interactive element
(e.g. an icon-only button), provide an `aria-label` on that element instead of relying on the
icon.
