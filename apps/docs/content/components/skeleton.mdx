## Props

### Skeleton

Accepts `React.HTMLAttributes<HTMLDivElement>`; shape and size are controlled entirely via `className` (e.g. `h-4 w-32`, `rounded-full`).

### Spinner

| Prop | Type | Default |
|------|------|---------|
| `size` | `"sm" \| "default" \| "lg"` | `"default"` |
| `className` | `string` | — |
