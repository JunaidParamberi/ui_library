## Usage

Use `Skeleton` to mirror the shape of the content that's loading (a `h-4 w-32` bar for a line
of text, `rounded-full` for an avatar) so the layout doesn't shift once real content arrives.
Use `Spinner` for indeterminate loading where there's no meaningful shape to mimic, such as a
button's `loading` state or a small inline fetch.

## Accessibility

Both are decorative and carry no text content, so screen readers announce nothing on their
own. Wrap the loading region in a container with `aria-busy="true"` (and `aria-live="polite"`
if content replaces it in place) so assistive tech knows a loading state is in progress.
