Shows contextual help text on hover or focus, built on Radix Tooltip.

## Props

`Tooltip`, `TooltipTrigger` are Radix `Root`/`Trigger` re-exports. `TooltipContent` extends Radix `Content` props (`sideOffset` defaults to `6`) plus `className`. Wrap usage in a `TooltipProvider`.
