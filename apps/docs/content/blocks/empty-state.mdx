## Usage

Use inside a `Card` or section body wherever a list, table, or search result is empty.
Pass a short `title` and one-line `description`, and include `action` (`{ label, onClick }`)
when there's a clear next step, such as creating the first item. Reserve the `icon` slot for
a single decorative Lucide icon that reinforces the empty context.

## Accessibility

The `title` renders as a heading so it participates in the page's heading hierarchy, and the
optional action is a real, keyboard-reachable button rather than a styled `div`. The
decorative icon is presentational and does not need alt text.
