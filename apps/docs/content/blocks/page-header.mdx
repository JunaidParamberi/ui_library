## Usage

Place at the top of a page, below any breadcrumb trail passed via `breadcrumbs`. Use
`description` for a one-line subtitle, `badge` for a status marker (e.g. a "Preview" or
"Beta" `Badge`), and `actions` for the page's primary and secondary buttons. Set `bordered`
to add a bottom rule when the header sits directly above dense content like a table.

## Accessibility

The `title` renders as the page's main heading, keeping the document outline predictable
for screen-reader users navigating by heading. Actions are ordinary buttons/links and
follow standard tab order after the breadcrumbs.
