## Usage

Pass `tiers` (`PricingTier[]`) with each tier's name, price, features, and call-to-action;
mark one tier `highlighted` to draw the eye to the recommended plan and optionally give it a
`badge` like "Popular". Set `columns` to match the number of tiers (2-4 reads best). Wire
`billingPeriod` as a controlled `{ value, onChange }` pair when tiers have both monthly and
annual pricing.

## Accessibility

Each tier's call-to-action is a real button, so the whole grid is keyboard-navigable via tab
order. The highlighted tier is distinguished visually and via its `badge` text, not by color
alone, so the recommendation is conveyed to assistive technology as well.
