## Usage

Compose the toolbar above a `DataTable` or `Table`. The `search` field is fully controlled;
add `filters` for one or more `Select`-backed filter dropdowns; and pass `actions` for
page-level controls like "Add member". When rows are selected, set `selectedCount` and
`bulkActions` to swap in a selection summary and bulk operations in place of the filters.

## Accessibility

Each filter `Select` receives an `aria-label` from its `placeholder` so filters remain
identifiable without visible labels. The search input uses `type="search"` for correct
assistive-technology semantics, and all controls are reachable via standard tab order.
