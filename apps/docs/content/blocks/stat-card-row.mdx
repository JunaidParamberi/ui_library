## Usage

Pass `stats` as a `KPI[]` (imported from `@manpowerhub/ui`) — each entry supplies a `label`,
`value`, and optional `delta`/`trend`/`sub` line. Set `columns` (1-4) to control how many
`KPICard`s appear per row before wrapping on smaller viewports.

## Accessibility

Each stat renders through `KPICard`, which pairs the numeric value with its label as
readable text rather than relying on layout alone, so screen readers announce a coherent
"label, value, trend" sequence for every card.
